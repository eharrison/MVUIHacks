## Installation

MVUIHacks is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "MVUIHacks", git: 'https://github.com/eharrison/MVUIHacks', tag: '0.1.8'
```
