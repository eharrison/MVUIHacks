## Installation

MVUIHacks is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "MVUIHacks", git: 'https://github.com/iOSWizards/MVUIHacks', tag: '0.2.2'
```
